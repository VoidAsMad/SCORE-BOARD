# Release_note

## 2022-05-29
> `/set [user] [point]` 으로 점수를 설정합니다.<br/>
> `/plus [user] [point]`으로 점수를 더합니다.<br/>
> `/minus [user] [point]`으로 점수를 차감합니다.<br/>
> `/point (user)`으로 점수를 확인합니다.<br/>
> `/initialization [user]`으로 점수를 초기화합니다.<br/>
